## Definition

Checklist-style recruitment videos explain a role or task through a numbered list. They work well when the viewer wants to avoid mistakes, prepare for work, or compare job expectations.

## Best Use Cases

- Onboarding tips
- First-day checklist
- Daily task standards
- Common mistakes
- Interview preparation

## Recommended Structure

| Stage | Time | Function |
| --- | --- | --- |
| Hook | 0-3s | State role + number + urgency |
| Value | 3-10s | Deliver checklist items quickly |
| Save Trigger | 10-13s | Ask if viewer remembered it |
| CTA | 13-16s | Follow, apply, or learn more |

## Production Notes

- Use real workplace visuals.
- Keep each item short.
- Use captions that accumulate visually.
- Avoid long intro scenes.
- Keep brand CTA after useful content.

## Reusable Formula

```text
[Role] + must-do [N] things + one cannot be missed
```

## Example Titles

- Front desk staff must do these 7 things
- Room attendants should never skip these 6 steps
- Restaurant supervisors must check these 5 points before opening
