# Account Report Sample

This is a sanitized sample. Account names and metrics are fictionalized.

## Account Overview

| Metric | Value |
| --- | --- |
| Industry | Hospitality |
| Positioning | Recruitment knowledge account |
| Followers | 1,200 |
| Total Likes | 8,600 |
| Content Focus | Job tips, role explainers, hiring pain points |

## Recent Content Diagnosis

| Content Type | Performance | Diagnosis |
| --- | --- | --- |
| Job checklist | Strong saves and shares | Useful and repeatable |
| Salary explainer | Strong comments | Triggers discussion |
| Pure hiring post | Weak engagement | Too direct, lacks content value |
| Workplace scenario | Medium engagement | Needs sharper opening hook |

## Strong Pattern

The best-performing videos usually put practical value before recruitment information.

```text
Useful job insight first
→ trust and attention
→ brand or job CTA later
```

## Risks

- Repetitive title format may fatigue viewers
- Pure recruitment posts underperform when no pain point is stated
- Comments are not yet being converted into new topic ideas

## Recommendations

- Use checklist and mistake-avoidance formats more often
- Put salary, schedule, and role growth points earlier
- Turn comments into follow-up topics
- Build a weekly topic matrix by role and pain point
